# moyai_samples

MoyaiのRemoteHeadを用いたサンプルゲームです。

## ソースコードの初期化

1. git clone moyai_sample
2. cd moyai_sample
3. git submodule update --init moyai

## サンプルゲームのビルド

1. moyai_sample/moyaiをビルドする(Moyaiのビルド手順に従う)
2. moyai_sample/Makefile で make する。  dm, rv, duelの3つが生成される。
